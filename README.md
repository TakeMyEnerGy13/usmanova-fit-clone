# UsmanovaFit — лендинг (репlica)

Реплика первых экранов лендинга [usmanovafit.gymteam.ru](https://usmanovafit.gymteam.ru/mainpage) — тестовое задание.

Воспроизведены **первые 4 блока + футер**:

1. **Hero** — сплит-лейаут с CTA.
2. **Доверие к тренеру** — карточка с буллетами + горизонтальная галерея фото.
3. **Социальное доказательство** — сетка медиа-плиток + CTA.
4. **Тренировки дома** — карточки программ (флагман + марафоны).
5. **FAQ** — рабочий аккордеон.
6. **Футер** — реквизиты, ссылки, соцсети.

## Что интерактивно

- **Кнопки** (`Выбрать программу`, `Вернуть форму`, `Подробнее`, `Оставить заявку`) открывают **модалку с формой заявки**.
- **Форма** — клиентская валидация (имя, телефон/e-mail, согласие) + экран успеха. Без бэкенда; точка интеграции с реальной отправкой помечена в `script.js`.
- **FAQ** — раскрывающийся аккордеон.

## Стек

Чистый **HTML + CSS + vanilla JS**, без фреймворков и сборки. Mobile-first, адаптив через один брейкпоинт (`768px`). Шрифт — Manrope (Google Fonts) как близкая замена Gilroy.

## Локальный запуск

Открыть `index.html` в браузере (или любой статический сервер).

## Примечание

Изображения взяты с оригинального лендинга для визуального соответствия в рамках тестовой реплики.
